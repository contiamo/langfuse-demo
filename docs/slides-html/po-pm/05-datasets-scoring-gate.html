Block B · The gate
The experiment & the gate
Change one variable model A vs B, or prompt v1 vs v2 — fixed dataset, fixed score
Compare side by side per-item faithfulness in the Experiments tab
Set the threshold 'within 5% → ship the cheaper model' — your call
The regression gate below threshold → CI blocks the merge
The threshold is a product decision — you own it.
